README (Bias Detector Extention)

Overview
The Bias Detector Extention is a browser extention that helps users identify and analyze
bias in online articles. This tool promotes media literacy by providing neutral, fact
based feedback.

Major feature
- Bias Highlighting: ability to scan news articles for biased language, load words, or
opinion presented as fact.

How it works: 
- Click exntention icon
- Bias Detector processes article information
- specific sentences and phrases are flagged
- Feedback isolates factual information

Supplemental Features

These features enhance core functionaliy and user value
- Alternative Source Aggregation: After complet scans, extention provides alternate news
articles from different outlets. Bias from primary article is now cross referenced for
perspective.
- Bias Explanation and Education: A section of is highlighted as bias, user can
- interact and recieve an explanation for the flagging.

Developer Instructions:
- CLone Respository
  git clone [respository_url]
-Install dependencies
  npm install (for web extention if needed)
  pip install -r requirements.txt
- Run local server
  pyton main.py (or similar command depending on API service)
- load extention in browser
  open browser extention maagment page
  enable "Developer Mode"
  click "Load Unpacked" and select extention root directory
